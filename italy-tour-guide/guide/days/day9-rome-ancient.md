## 历史背景：罗马帝国的崛起与衰落

### 从小村庄到世界帝国

**罗马城传说建于公元前753年4月21日，从一个河边的小村庄发展成统治地中海的帝国。**

**历史时间线：**

| 时期 | 年份 | 事件 |
|------|------|------|
| 王政时代 | 753-509 BC | 七位国王统治 |
| 共和国 | 509-27 BC | 元老院统治，扩张到地中海 |
| 帝国早期 | 27 BC-284 AD | 奥古斯都到戴克里先 |
| 帝国晚期 | 284-476 AD | 东西分治，西罗马衰落 |
| 灭亡 | 476 AD | 西罗马帝国灭亡 |

### 罗马工程天才

**罗马人不是艺术家，而是工程师。他们发明了：**

#### 1. 混凝土（Concrete）

**罗马混凝土用火山灰、石灰、碎砖混合，比现代混凝土更耐久。**

**为什么万神庙保存2000年？** 因为罗马混凝土。

#### 2. 拱券（Arch）

**拱券可以把重量分散到柱子上，让建筑更宽敞。**

**应用：**

- 渡槽（送水）
- 斗兽场
- 万神庙
- 凯旋门

#### 3. 穹顶（Dome）

**万神庙的穹顶是古代世界最大的，1400年后才被佛罗伦萨大教堂超越。**

#### 4. 道路系统

**"条条大路通罗马"不是空话——罗马帝国修建了40万公里道路，其中8万公里铺石。**

**技术：**

- 多层结构（沙石、碎石、石板）
- 路面拱起（排水）
- 每罗马里（1.5公里）一个里程碑

### 罗马社会

**罗马社会等级森严：**

| 等级 | 人口比例 | 特权 |
|------|----------|------|
| 元老院贵族 | <1% | 政治权力 |
| 骑士阶层 | ~2% | 商业、税收 |
| 平民 | ~90% | 公民权 |
| 解放奴隶 | ~5% | 有限权利 |
| 奴隶 | ~10-20% | 无权利 |

**面包与马戏（Panem et Circenses）：**

罗马皇帝用免费粮食和角斗士比赛安抚民众。斗兽场就是这种政策的产物。

### 古代世界的中心

**古罗马广场是罗马共和国和帝国的政治、宗教、商业中心。2000年前，这里是世界的"神经中枢"。**

**历史：**

| 时期 | 功能 |
|------|------|
| 王政时代 | 墓地 |
| 共和国 | 政治、商业、宗教中心 |
| 帝国 | 更加热闹，建造更多神庙 |
| 中世纪 | 被遗弃，变成牧场 |
| 文艺复兴 | 大理石被掠夺 |
| 19世纪 | 开始考古发掘 |

### 主要看点

#### 1. 售票处与入口

**8点到达古罗马广场售票处（Biglietteria Foro Romano），排队领免费票。**

**位置：** 在Via di San Gregorio街上，靠近斗兽场。

#### 2. 提图斯凯旋门（Arco di Tito）

**入口处的凯旋门，纪念公元70年提图斯皇帝征服耶路撒冷。**

**看点：**

- 内侧浮雕：罗马士兵带着战利品离开耶路撒冷
- 七臂烛台（Menorah）：从犹太圣殿掠夺
- 这是罗马现存最古老的凯旋门（81年）

**历史意义：** 这座凯旋门记录了犹太历史上最悲惨的事件之一——第二圣殿被毁。

#### 3. 圣道（Via Sacra）

**广场中央的主要道路，古代举行凯旋游行的地方。**

**想象：** 将军乘战车，展示俘虏和战利品，人群欢呼。

#### 4. 农神庙（Tempio di Saturno）

**广场最显眼的遗迹，8根柱子依然矗立。**

**历史：**

- 公元前497年：最早的神庙
- 283年：重建（现存遗迹）
- 功能：国库所在地

**农神节：** 每年12月17-23日庆祝，是圣诞节的源头之一。

#### 5. 维斯塔神庙（Tempio di Vesta）

**圆形小神庙，供奉灶神维斯塔。**

**维斯塔贞女（Vestal Virgins）：**

- 6名女祭司，6-10岁入选
- 必须守贞30年
- 失贞者被活埋
- 她们守护圣火，如果熄灭，罗马会灭亡

**看点：** 圆形建筑（模仿早期罗马茅屋）。

#### 6. 维斯塔贞女之家（Casa delle Vestali）

**维斯塔贞女的住所，在神庙旁边。**

**看点：**

- 庭院
- 雕像基座（原来是维斯塔贞女的雕像）
- 水池

#### 7. 凯撒神庙（Tempio del Divo Giulio）

**纪念被刺杀的凯撒。**

**历史：** 公元前44年3月15日，凯撒在这里被刺杀（准确地点在附近的一个会议室）。他死后被火化，骨灰放在这里。

**看点：：** 祭坛上还有人们献的花。

#### 8. 塞维鲁凯旋门（Arco di Settimio Severo）

**广场北端的巨大凯旋门（203年），纪念塞维鲁皇帝战胜帕提亚人。**

**看点：**

- 三通道
- 浮雕描绘战争场景
- 顶部的铭文原本提到塞维鲁的两个儿子，但一个儿子后来被杀，名字被抹去

#### 9. 元老院（Curia）

**保存最好的古罗马建筑之一，是元老院开会的场所。**

**历史：**

- 最初建于公元前7世纪
- 283年重建（现存建筑）
- 630年改为教堂（所以保存完好）

**看点：** 内部可以看到古代的地面和大理石装饰。

#### 10. 佛卡圆柱（Colonna di Foca）

**广场中央的孤立柱子，608年为纪念拜占庭皇帝佛卡而立。**

**意义：** 它是广场上最后添加的纪念碑，标志着广场的衰落。

### 罗马的诞生地

**帕拉丁山是罗马七丘中最中心的一座，也是罗马城的起源地。**

**神话：**

传说罗慕路斯和雷穆斯兄弟被遗弃在台伯河边，被母狼哺育。后来罗慕路斯在帕拉丁山建立了罗马城（公元前753年）。

**历史：**

- 铁器时代：最早的定居点
| 共和国 | 贵族住宅区 |
| 帝国 | 皇帝宫殿所在地 |

**词源：** 英语"palace"（宫殿）来自"Palatine"。

### 主要看点

#### 1. 皇帝宫殿（Domus Augustana）

**多米提安皇帝（81-96年）建造的巨大宫殿群。**

**看点：**

- 宴会厅
- 私人公寓
- 庭院
- 体育场

#### 2. 利维亚之家（Casa di Livia）

**奥古斯都皇帝的妻子利维亚的住所。**

**看点：** 精美的壁画（现在在罗马国家博物馆）。

#### 3. 罗慕路斯小屋

**铁器时代的小屋遗迹，传说是罗慕路斯的住所。**

#### 4. 观景台

**从帕拉丁山可以俯瞰古罗马广场和斗兽场。**

### 古代世界最大的圆形剧场

**斗兽场高48.5米，长189米，宽156米，可容纳5万-8万观众，是古代世界最大的圆形剧场。**

**历史：**

| 年份 | 事件 |
|------|------|
| 72年 | 韦帕芗皇帝开始建造 |
| 80年 | 提图斯皇帝完工 |
| 80年 | 开幕100天庆典，杀死9000只动物 |
| 404年 | 角斗士比赛被禁止 |
| 523年 | 动物狩猎被禁止 |
| 中世纪 | 被当作采石场 |
| 19世纪 | 开始保护 |

**建造者：**

- 韦帕芗（Vespasian, 69-79年在位）：开始建造
- 提图斯（Titus, 79-81年在位）：完工
- 钱从哪来？公元70年征服耶路撒冷掠夺的财富，由犹太奴隶建造。

### 建筑奇迹

**斗兽场是罗马工程技术的巅峰。**

#### 1. 混凝土与拱券

**斗兽场用了80个拱门支撑上层结构，可以快速疏散观众。**

**计算：** 5万观众可以在15分钟内全部撤离。

#### 2. 座位分配

**座位按社会等级分配：**

| 位置 | 观众 |
|------|------|
| 底层（ podium） | 皇帝、元老院、维斯塔贞女 |
| 二层 | 骑士阶层 |
| 三层 | 普通公民 |
| 四层 | 女性奴隶、穷人 |

#### 3. 遮阳篷（Velarium）

**夏天，水手会在顶部拉起巨大的遮阳篷，保护观众免受日晒。**

#### 4. 地下迷宫（Hypogeum）

**斗兽场地下有一个复杂的迷宫：**

- 野兽笼子
- 电梯（80个！）
- 角斗士等候室
- 储藏室

**效果：** 野兽和角斗士可以从地下突然出现，制造戏剧效果。

### 角斗士比赛

**角斗士比赛起源于葬礼仪式，后来演变成大众娱乐。**

#### 角斗士类型

| 类型 | 武器 | 特点 |
|------|------|------|
| Murmillo | 剑、盾、头盔 | 像罗马士兵 |
| Retiarius | 三叉戟、网 | 像渔夫 |
| Thraex | 弯刀、小盾 | 像色雷斯人 |
| Secutor | 剑、光滑头盔 | 专门对付Retiarius |

#### 比赛流程

1. **上午**：动物狩猎（venationes）
2. **中午**：处决犯人
3. **下午**：角斗士比赛

**残酷的娱乐：**

- 开幕100天杀了9000只动物
- 有些动物因此灭绝（北非象）
- 角斗士死亡率约20%（每5场比赛死1人）

**神话破除：**

- 不存在"拇指朝下=死"的手势（这是好莱坞发明）
- 皇帝很少干涉比赛结果
- 角斗士不是全是奴隶（有些是自由人，为了钱和名声）

### 基督教殉道者？

**传说：** 基督徒被扔进狮子笼。

**真相：** 历史学家没有找到可靠证据。斗兽场主要是角斗士比赛和动物狩猎，不是迫害基督徒的地方。

**但教会传统认为：** 16世纪，教皇本笃十四世宣布斗兽场为基督徒殉道圣地，禁止继续破坏。

### 参观路线

**推荐路线（1-1.5小时）：**

1. 一楼：看整体结构
2. 二楼：看座位区和地下迷宫
3. 展览：了解角斗士和建筑技术

**最佳拍照：**

- 外观：从地铁站方向拍全景
- 内部：从上层看地下迷宫

### 撒谎者的测试

**真理之口是一块巨大的圆形石盘，上面雕刻着河神或特里同的面孔，嘴里有一个洞。**

**传说：** 如果把手放进嘴里并说谎，手会被咬掉。

**真实历史：**

- 这是古代的井盖或喷泉装饰（可能是1世纪）
- 1632年放在圣母教堂（Santa Maria in Cosmedin）的门廊
- 1953年电影《罗马假日》让它闻名世界

**测试方法：**

1. 把手放进嘴里
2. 说一句话
3. 如果没被咬，就是真话

**趣闻：** 排队的人很多，但很快。

### 意大利统一的象征

**维托里奥·埃马努埃莱二世纪念碑（Il Vittoriano）建于1885-1935年，纪念意大利统一和第一位国王。**

**特点：**

- 纯白色大理石（来自 Botticino）
- 宽135米，高70米
- 气势恢宏

**争议：**

- 罗马人叫它"打字机"或"结婚蛋糕"（认为太浮夸）
- 与周围古建筑风格不协调
- 但现在已成为罗马标志

**主要部分：**

#### 1. 无名烈士墓

**纪念一战中牺牲的无名士兵，有卫兵站岗。**

#### 2. 维托里奥二世铜像

**12米高的国王骑马铜像。**

#### 3. 统一博物馆

**展示意大利统一的历史。**

#### 4. 登顶观景台

**可以俯瞰整个罗马。**

### 威尼斯广场

**纪念碑前面的广场，因威尼斯宫（Palazzo Venezia）得名。**

**趣闻：** 墨索里尼的办公室在威尼斯宫，他曾从阳台上向人群演讲。

## 今日总结

今天你走过了2000年前的罗马：古罗马广场曾经是世界的中心，元老院在这里辩论，凯撒在这里演讲；斗兽场曾经是5万观众的娱乐场，角斗士在这里流血；帕拉丁山曾经是皇帝的宫殿。罗马帝国已经消失，但混凝土、拱券、道路系统——这些遗产改变了世界。
