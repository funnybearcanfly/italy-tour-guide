### 城市背景：中世纪的海上强国

比萨今天只是个大学城，但在中世纪，它是与威尼斯、热那亚齐名的海上共和国。

**历史时间线：**

| 时期 | 年份 | 事件 |
|------|------|------|
| 古罗马 | 公元前2世纪 | 罗马殖民地 |
| 中世纪 | 11-13世纪 | 海上共和国黄金时代 |
| 中世纪 | 1063年 | 舰队击败阿拉伯人，掠夺大量财富 |
| 中世纪 | 1173年 | 开始建造钟楼（斜塔） |
| 衰落 | 1284年 | 梅洛里亚海战败给热那亚 |
| 衰落 | 1406年 | 被佛罗伦萨征服 |
| 现代 | 1987年 | 奇迹广场成为UNESCO世界遗产 |

**为什么比萨衰落？**

- 阿诺河淤积，港口失去功能
- 海战失败
- 佛罗伦萨的崛起

### 奇迹广场（Piazza dei Miracoli）

奇迹广场的名字来自意大利作家加布里埃尔·邓南遮（Gabriele D'Annunzio），他称这里是"prato dei Miracoli"（奇迹的草地）。

**四大奇迹：**

1. **比萨大教堂**（Duomo di Pisa）
2. **比萨斜塔**（Torre pendente di Pisa）
3. **洗礼堂**（Battistero）
4. **墓园**（Camposanto Monumentale）

**建筑风格：比萨罗马式（Pisan Romanesque）**

- 白色大理石
- 盲拱装饰
- 层层叠叠的拱廊
- 伊斯兰和拜占庭元素

### 比萨斜塔

#### 建造历史：一个"美丽"的错误

| 年份 | 事件 |
|------|------|
| 1173年8月9日 | 开始建造（建筑师Bonanno Pisano） |
| 1178年 | 建到第4层时开始倾斜，工程暂停 |
| 1272年 | 恢复建造，试图修正倾斜（结果更斜） |
| 1372年 | 完工（共199年！） |
| 1990年 | 因倾斜过度关闭 |
| 2001年 | 修复后重新开放 |

**为什么会斜？**

- 地基只有3米深
- 下面的土壤松软（黏土、沙子混合）
- 塔身南边的土壤比北边更软

**数据：**

- 高度：55.86米（从地面到低侧）
- 重量：约14,500吨
- 倾斜：约4度（从垂直线偏离约4米）
- 台阶：296级

#### 伽利略的传说

**传说：** 伽利略从斜塔上扔下两个不同重量的球，证明它们同时落地。

**真相：** 很可能是虚构的。伽利略本人从未在著作中提到这个实验，这个故事来自他的学生维维安尼的传记。

但伽利略确实出生在比萨（1564年），也确实在比萨大学学习过。

### 城市背景：文艺复兴的真正发源地

如果说罗马是古文明的中心，米兰是商业重镇，那么佛罗伦萨就是文艺复兴的灵魂。

#### 佛罗伦萨的黄金时代

**佛罗伦萨（Firenze）的名字来自拉丁语"florentia"，意为"花城"。**

**历史时间线：**

| 时期 | 年份 | 事件 |
|------|------|------|
| 古罗马 | 公元前59年 | 凯撒建立殖民点 |
| 中世纪 | 1115年 | 成为自治城邦 |
| 中世纪 | 1252年 | 铸造金币"弗罗林"（fiorino），欧洲通用货币 |
| 文艺复兴 | 1434年 | 美第奇家族掌权 |
| 文艺复兴 | 1469-1492年 | 洛伦佐·德·美第奇"伟大的"统治 |
| 衰落 | 1494年 | 法军入侵，美第奇流亡 |
| 统一 | 1865-1871年 | 意大利王国首都 |

#### 美第奇家族：文艺复兴的教父

**没有美第奇家族，就没有文艺复兴。** 这句话一点不夸张。

**家族起源：**

美第奇（Medici）原本是农民，13世纪搬到佛罗伦萨，从事羊毛贸易和银行业。到15世纪，他们成为欧洲最富有的家族，甚至借钱给国王和教皇。

**关键人物：**

##### 科西莫·德·美第奇（Cosimo de' Medici, 1389-1464）

- 1434年成为佛罗伦萨的实际统治者（名义上是普通公民）
- 创立了柏拉图学院
- 资助多纳泰罗、布鲁内莱斯基等艺术家
- 被称为"国父"（Pater Patriae）

##### 洛伦佐·德·美第奇（Lorenzo de' Medici, 1449-1492）

- 外号"伟大的"（Il Magnifico）
- 资助米开朗基罗、波提切利、达·芬奇
- 自己就是诗人和人文主义者
- 统治时期是佛罗伦萨的黄金时代

> **趣闻：** 米开朗基罗年轻时住在美第奇宫殿里，和洛伦佐的儿子们一起长大，被称为"我们的孩子"。

##### 教皇利奥十世（Pope Leo X, 1475-1521）

- 洛伦佐的儿子，原名乔瓦尼·德·美第奇
- 1513年当选教皇
- 他说过"上帝给了我们教皇的位置，让我们好好享受"
- 大力资助艺术，但也导致了新教改革

##### 凯瑟琳·德·美第奇（Catherine de' Medici, 1519-1589）

- 嫁给法国国王亨利二世
- 把意大利的烹饪、时尚、艺术带到法国
- 法国宫廷的"幕后女王"

**家族遗产：**

- 两位教皇（利奥十世、克莱门特七世）
- 两位法国王后（凯瑟琳、玛丽）
- 无数艺术赞助
- 乌菲兹美术馆的藏品原本是他们的收藏

#### 为什么佛罗伦萨孕育了文艺复兴？

**经济基础：**

- 羊毛和银行业带来巨大财富
- 弗罗林金币是欧洲通用货币
- 商人阶层有钱有闲

**社会结构：**

- 城邦政治，贵族与商人合作
- 相对开放的社会氛围
- 公民参与公共事务

**文化氛围：**

- 拜占庭学者带来古希腊典籍（1453年君士坦丁堡陷落后）
- 人文主义思想流行
- 艺术家地位提高

**美第奇的作用：**

- 科西莫说"花钱买美丽，永远不会后悔"
- 艺术是政治声望的工具
- 与艺术家建立私人友谊

#### 佛罗伦萨的标志

老桥是佛罗伦萨唯一在二战中幸存的桥，也是欧洲最古老的石造封闭桥梁。

**历史：**

- 972年首次记载
- 1345年重建（现存的桥）
- 1593年费迪南多一世下令只能开珠宝店

**为什么桥上有房子？**

中世纪，桥上建房很常见。老桥上的店铺原本是肉铺和鱼铺（可以直接把废料扔进河里），但太臭了，大公下令改成珠宝店。

**瓦萨里走廊（Corridoio Vasariano）：**

桥上有一条高架走廊，连接乌菲兹美术馆和皮蒂宫。1565年，科西莫一世大公建造这条走廊，这样他可以安全地从政府办公室走到私人宫殿，不用和普通人混在一起。

#### 佛罗伦萨的政治心脏

领主广场从中世纪起就是佛罗伦萨的政治中心。

**主要建筑：**

- **旧宫（Palazzo Vecchio）**: 1299年建造，至今仍是市政府
- **兰齐凉廊（Loggia dei Lanzi）**: 户外雕塑馆
- **乌菲兹美术馆**: 原本是政府办公楼

**雕塑：**

- 大卫像复制品（原作在学术美术馆）
- 海神喷泉
- 切利尼《珀耳修斯手持美杜莎头颅》
- 詹博洛尼亚《萨宾妇女被劫》

#### 意大利的"先贤祠"

圣十字教堂（Basilica di Santa Croce）被称为"意大利的先贤祠"，因为这里安葬着：

- 米开朗基罗

**趣闻：** 教堂地板上满是墓碑，走在上面要小心。
