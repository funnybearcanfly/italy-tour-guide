## 人物故事：米开朗基罗的罗马岁月

### 从雕塑家到画家

米开朗基罗·博那罗蒂（Michelangelo Buonarroti, 1475-1564）的一生可以分为三个阶段：

| 时期 | 地点 | 主要作品 |
|------|------|----------|
| 早期（1475-1505） | 佛罗伦萨 | 《哀悼基督》《大卫》 |
| 中期（1505-1534） | 罗马 | 西斯廷天花板、摩西像 |
| 晚期（1534-1564） | 罗马 | 《最后的审判》、圣彼得大教堂穹顶 |

**1505年，教皇尤利乌斯二世（Julius II）召唤30岁的米开朗基罗到罗马，委托他设计一座宏大的陵墓。**

**米开朗基罗的梦想：** 这是他一生中最想完成的项目——一座有40尊雕像的巨型陵墓。

**现实：** 教皇很快改变了主意，让他去画西斯廷礼拜堂的天花板。

**米开朗基罗的抱怨：** "我是雕塑家，不是画家！" 但他不得不服从。

### 西斯廷礼拜堂的痛苦

**1508-1512年，米开朗基罗花了4年画西斯廷天花板。**

**工作条件：**

- 站在脚手架上（不是躺着！）
- 脖子仰着
- 颜料滴在脸上
- 没有助手（他解雇了所有帮他的人）

**他写给朋友的诗：**

> "我的腰像弓一样弯着，
> 我的屁股像马屁股，
> 我的颜料滴在脸上……
> 我已经不成人形了。"

**结果：** 他创造了西方艺术史上最伟大的作品。

### 《最后的审判》的痛苦

**1536-1541年，61岁的米开朗基罗花了5年画《最后的审判》。**

**背景：** 1527年罗马被洗劫，1534年新教改革冲击天主教。教会要求米开朗基罗创作一幅"警告"世人的作品。

**特点：** 与天花板的秩序不同，《最后的审判》充满混乱和恐惧——反映了米开朗基罗晚年的宗教焦虑。

**自画像：** 米开朗基罗把自己的脸画在圣巴多罗买手中被剥下的人皮上——表达他的痛苦和负罪感。

### 世界上最小的国家，最大的博物馆

**梵蒂冈城国（Stato della Città del Vaticano）面积只有0.44平方公里，人口约800人，是世界上最小的国家。**

**但它的博物馆收藏了人类历史上最伟大的艺术。**

**历史：**

| 年份 | 事件 |
|------|------|
| 1506年 | 教皇尤利乌斯二世创建博物馆（展示拉奥孔雕像） |
| 1771年 | 向公众开放 |
| 现代 | 每年接待600万游客，世界第三大博物馆 |

### 参观路线

**梵蒂冈博物馆很大（7公里走廊！），需要规划路线。**

**推荐路线（3-4小时）：**

1. 入口 → 螺旋楼梯（拍照）
2. 庇护-克莱门蒂诺博物馆（Museo Pio-Clementino）：古代雕塑
3. 挂毯画廊（Galleria degli Arazzi）：16世纪挂毯
4. 地图画廊（Galleria delle Carte Geografiche）：16世纪意大利地图
5. 拉斐尔房间（Stanze di Raffaello）：拉斐尔的壁画
6. **西斯廷礼拜堂（Cappella Sistina）**：米开朗基罗的杰作

### 主要看点详解

#### 1. 拉奥孔群像（Gruppo del Laocoonte）

**1506年在罗马出土，教皇尤利乌斯二世立即购买，成为梵蒂冈博物馆的第一件藏品。**

**故事：**

特洛伊祭司拉奥孔警告特洛伊人"不要把木马带进城"，但雅典娜（支持希腊人的神）派海蛇咬死他和他的两个儿子。

**艺术价值：**

- 公元前1世纪的希腊原作（或罗马复制）
- 表现极度痛苦
- 影响了米开朗基罗的雕塑风格

#### 2. 阿波罗·贝尔维德尔（Apollo del Belvedere）

**古代最著名的雕像之一，18-19世纪被认为是"完美的男性美"。**

**影响：** 启发了无数新古典主义艺术家。

#### 3. 地图画廊（Galleria delle Carte Geografiche）

**120米长的走廊，墙上画着40幅意大利地图（1580-1583年）。**

**看点：**

- 教皇格里高利十三世委托
- 16世纪对意大利地理的认知
- 天花板也是华丽的壁画

#### 4. 拉斐尔房间（Stanze di Raffaello）

**教皇尤利乌斯二世的私人公寓，由拉斐尔和他的学生装饰（1508-1524年）。**

##### 签字厅（Stanza della Segnatura）

**最重要的房间，原是教皇的图书馆。**

**四幅主要壁画代表四种智慧：**

1. **《雅典学院》（Scuola di Atene, 1509-1511）**：哲学

   **这是拉斐尔最著名的作品。**

   **画面描述：**

   - 古代哲学家聚集在宏伟的建筑中
   - 中央：柏拉图（指着天）和亚里士多德（指着地）
   - 左边：苏格拉底在争论
   - 前景：赫拉克利特（米开朗基罗的脸！）
   - 右下角：拉斐尔的自画像

   **趣闻：**

   - 柏拉图的脸是达·芬奇
   - 赫拉克利特是米开朗基罗（当时米开朗基罗正在隔壁画西斯廷）
   - 欧几里得的脸是建筑师布拉曼特

2. **《圣礼之争》（Disputa del Sacramento）**：神学
3. **《帕纳苏斯山》（Parnaso）**：诗歌
4. **《三德》（Le Virtù）**：法律

##### 赫利奥多罗斯厅（Stanza di Eliodoro）

**描绘上帝保护教会的故事。**

**重点作品：**

- 《赫利奥多罗斯被逐出圣殿》
- 《波尔塞纳的弥撒》

#### 5. 西斯廷礼拜堂（Cappella Sistina）

**西方艺术史上最重要的房间。**

##### 历史

**1473-1481年，教皇西斯廷四世建造这座礼拜堂，以自己名字命名。**

**功能：**

- 教皇的私人礼拜堂
- 红衣主教选举新教皇的地方（conclave）

##### 天花板：《创世纪》（1508-1512）

**米开朗基罗4年的心血，共9幅中央画，描绘《创世记》的故事。**

**9幅中央画（从祭坛方向）：**

1. 光与暗的分离
2. 日月的创造
3. 陆地与水的分离
4. 亚当的创造
5. 夏娃的创造
6. 原罪与被逐出伊甸园
7. 诺亚的献祭
8. 大洪水
9. 诺亚醉酒

**最著名：** 《创造亚当》——上帝和亚当的手指几乎触碰，中间的空白是最著名的"未完成瞬间"。

**解读：**

- 上帝在云中，周围有粉色的披风（可能是夏娃和未来的孩子）
- 亚当慵懒地伸出手
- 两根手指之间的空白——"创造的火花"

**其他部分：**

- **先知和女巫**：12个巨大人物，预言基督
- **基督祖先**：三角拱中的小画
- **角落裸男**（Ignudi）：装饰性裸体男性

**趣闻：**

- 米开朗基罗最初拒绝，说"我是雕塑家"
- 他解雇了所有助手，自己画大部分
- 工作时脖子永远仰着，后来背驼了

##### 祭坛墙：《最后的审判》（1536-1541）

**25年后，61岁的米开朗基罗花了5年画这幅巨型壁画（1370×1220厘米）。**

**画面描述：**

- 中央：基督审判世人
- 左边（上升）：被拯救的灵魂升入天堂
- 右边（下降）：被诅咒的灵魂堕入地狱
- 底部：死人复活

**看点：**

- 基督不是温和的，而是愤怒的
- 圣母玛利亚躲在他身边
- 圣巴多罗买手持自己被剥下的人皮（上面有米开朗基罗的自画像！）

**趣闻：**

- 原本所有人物都是裸体的
- 1564年（米开朗基罗去世后），教皇命令丹尼尔·达·沃尔泰拉给裸体人物画上"裤子"（被称为"裤子制造者"braghettone）
- 1994年修复时，保留了这些"裤子"

##### 四周的壁画

**西斯廷礼拜堂的墙壁也有15世纪著名画家的壁画：**

- **北墙**：基督生平（波提切利、吉兰达约等）
- **南墙**：摩西生平（波提切利、吉兰达约等）

**看点：** 波提切利的《基督的试探》和《摩西的试探》。

### 世界上最大的教堂

**圣彼得大教堂长186米，高136米（穹顶），可容纳6万人，是世界上最大的教堂。**

**为什么在这里？**

传统认为，圣彼得（耶稣的门徒、第一任教皇）在公元64-67年尼禄迫害基督徒期间被倒钉十字架殉道，埋葬在这里。

**历史：**

| 年份 | 事件 |
|------|------|
| 324年 | 君士坦丁大帝建造第一座圣彼得教堂 |
| 1506年 | 教皇尤利乌斯二世决定重建 |
| 1547年 | 米开朗基罗接手设计（72岁！） |
| 1590年 | 穹顶完成 |
| 1626年 | 教堂祝圣 |

**建筑师接力：**

- 布拉曼特（Bramante, 1506-1514）
- 拉斐尔（Raphael, 1514-1520）
- 米开朗基罗（Michelangelo, 1547-1564）
- 卡洛·马德诺（Carlo Maderno, 1607-1629，立面）
- 贝尔尼尼（Bernini, 1629-1667，内部装饰）

### 内部看点

#### 1. 米开朗基罗《哀悼基督》（Pietà, 1498-1499）

**这是米开朗基罗23岁的作品，也是他唯一签名的作品。**

**故事：**

米开朗基罗听到有人说这不是他雕的，一怒之下在圣母的肩带上刻下了"MICHAELA[N]GELUS BONAROTUS FLORENTIN[US] FACIEBAT"（佛罗伦萨人米开朗基罗·博那罗蒂制作）。后来他后悔了，再也没签过名。

**看点：**

- 圣母年轻得不像基督的母亲（米开朗基罗解释：圣母的纯洁让她不会衰老）
- 基督躺在圣母腿上，像睡着了
- 衣褶的精细
- 圣母的表情（悲伤但平静）

**1972年的袭击：**

一个匈牙利人用锤子砸坏了圣母的手臂和鼻子。修复后，现在用防弹玻璃保护。

#### 2. 贝尔尼尼青铜华盖（Baldacchino, 1624-1633）

**贝尔尼尼26米高的青铜华盖，矗立在圣彼得的墓穴上方。**

**特点：**

- 四根螺旋柱
- 青铜来自万神庙（有争议）
- 顶部有地球仪和十字架

**趣闻：** 柱子上的蜜蜂是教皇乌尔班八世（巴贝里尼家族）的标志。

#### 3. 圣彼得宝座（Cattedra di San Pietro）

**贝尔尼尼1657-1666年设计的金色宝座，据说包含圣彼得真正坐过的椅子。**

**看点：**

- 四位教父支撑宝座
- 金色光芒（代表圣灵）

#### 4. 圣彼得铜像

**13世纪的铜像，据说摸他的脚会带来好运。**

**结果：** 脚趾被摸秃了。

### 贝尔尼尼的杰作

**1656-1667年，贝尔尼尼设计了圣彼得广场，像"母亲的双臂"拥抱世界。**

**数据：**

- 长：320米
- 宽：240米
- 柱子：284根
- 雕像：140尊圣人

**方尖碑：**

中央的方尖碑高25.5米，来自古埃及（公元前1世纪被带到亚历山大，37年被带到罗马）。

**趣闻：** 1586年移动方尖碑时，教皇下令"不许说话"，违者死刑。一个水手发现绳子冒烟，大喊"给绳子浇水！"——他不仅没被处死，还得了奖赏。

### 从陵墓到堡垒

**圣天使堡的历史几乎是罗马历史的缩影。**

| 时期 | 功能 |
|------|------|
| 123-139年 | 哈德良皇帝的陵墓 |
| 401年 | 军事堡垒 |
| 14世纪 | 教皇避难所 |
| 16世纪 | 监狱 |
| 现代 | 博物馆 |

**名字来源：**

590年，教皇格里高利一世看到天使在城堡顶上收剑入鞘，象征瘟疫结束。于是城堡改名"圣天使堡"。

### 圣天使桥

**通往城堡的桥，两旁有10尊天使雕像（贝尔尼尼工作室制作）。**

**看点：**

- 每位天使手持基督受难的一件器具（十字架、钉子、荆棘冠等）
- 两尊是贝尔尼尼原作（现在在Santa Maria della Pietà教堂，桥上是复制品）

## 今日总结
